PHOTO INSTRUCTIONS
==================
1. Add your photos here (JPG, PNG, or WEBP).
2. Then open script.js and update the photoData array at the top:

   const photoData = [
     { src: "assets/images/photo1.jpg", caption: "Our first memory 💕" },
     { src: "assets/images/photo2.jpg", caption: "That perfect day 🌸"  },
     { src: "assets/images/photo3.jpg", caption: "Always laughing ❤️"  },
     // add more...
   ];

3. If a src is empty (""), a decorative placeholder card is shown instead.

Recommended photo size: 800×800 px or square crop works best.
